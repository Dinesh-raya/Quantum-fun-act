This project is an educational game built with Streamlit and Qiskit designed to teach basic quantum concepts in a kid-friendly way.
Everything runs locally using Qiskit's simulator (no cloud APIs required).

## What's new in this version
- Animated Teleportation mini-game (step-by-step scenes) with interactive corrections.
- XP system + badges + Level Unlock Map (progression).
- SVG character cast: Spinny, Tango, Grov, Bitty with simple animations.
- Auto-demo algorithms: Deutsch-Jozsa, Grover, Teleportation.
- Bloch-sphere preview when Plotly + statevector simulator available.

## Run locally
1. Create a Python environment
2. `pip install -r requirements.txt`
3. `streamlit run app.py`

Enjoy! This is designed to be kid-friendly and fun.
